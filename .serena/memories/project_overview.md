# Foodshare iOS Project Overview

## Purpose
Foodshare is a food sharing platform connecting people with surplus food to those who need it. The iOS app enables users to browse, create, and claim food listings in their local area.

## Tech Stack
- **Language**: Swift 6.2, iOS 17.0+
- **UI**: SwiftUI with Liquid Glass Design System
- **Architecture**: MVVM + Clean Architecture (Feature-Based)
- **State Management**: `@Observable` macro
- **Backend**: Supabase (PostgreSQL + PostGIS, Auth, Storage, Realtime)
- **Dependencies**: supabase-swift 2.37.0, Kingfisher, Lottie

## Key Features
- Food listing browsing and creation
- Location-based discovery (Map view)
- Real-time messaging between users
- User profiles and ratings
- Community forum
- Search with filters
- Authentication (email, social)

## Project Version
Version 3.0.0 - Production Ready
